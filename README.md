## 🎮 Higher Lower Game (Python)

### 📌 Overview
- Console-based Higher-Lower game  
- Inspired by social media follower comparisons  
- Uses dataset based on accounts from **2020**  
- Built using core Python concepts  

### 🚀 Features
- 🔁 Random account selection  
- 👥 Compare two profiles (A vs B)  
- 🧠 User guesses who has more followers  
- ✅ Score tracking system  
- ❌ Game ends on wrong answer  
- 🧹 Clears screen using `print("\n" * 100)`  
- 🎨 ASCII art for better UI  

### 🛠️ Tech Used
- Python  
- Random module  
- Functions & loops  
- Dictionaries  

### 📂 Project Structure
- `main.py` → Game logic  
- `game_data.py` → Dataset (2020 follower data)  
- `art.py` → ASCII visuals  

### ▶️ How to Run
```bash
python main.py
